HTML5

HTML5 adalah versi terbaru dari HTML.

Ini adalah bahasa standar untuk membuat halaman web.

HTML5 mendeskripsikan struktur halaman web.

Elemen-elemen diwakili oleh tag-tag.

Contoh elemen HTML5:

<article>Ini adalah sebuah artikel</article>

Tag HTML tidak case sensitive.

Elemen HTML5 yang umum meliputi:

  • <section> untuk bagian
  • <header> untuk header
  • <footer> untuk footer

Dokumen HTML5 dimulai dengan deklarasi doctype:

<!DOCTYPE html>
CSS3

CSS3 adalah versi terbaru dari CSS.

Ini digunakan untuk mengatur tampilan dan tata letak halaman web.

CSS3 mengontrol warna, font, dan spasi elemen.

Contoh aturan CSS3:

body { background-color: lightgreen; }

CSS3 dapat ditambahkan ke HTML dengan tiga cara:

  • Inline - menggunakan atribut style
  • Internal - menggunakan elemen <style> di bagian head
  • Eksternal - menggunakan file CSS eksternal

Contoh tautan ke file CSS eksternal:

<link rel="stylesheet" href="styles.css">
JavaScript ES6

JavaScript ES6 adalah versi terbaru dari JavaScript.

Digunakan untuk membuat halaman web interaktif.

JavaScript dapat memperbarui dan mengubah HTML dan CSS.

Dapat digunakan untuk memvalidasi data dan membuat konten dinamis.

Contoh fungsi JavaScript:

const greet = () => {
  console.log("Hello, World!");
}

JavaScript adalah bahasa sisi klien.

Event JavaScript yang umum meliputi:

  • onclick
  • onmouseover
  • onmouseout

JavaScript dapat ditautkan ke dokumen HTML menggunakan tag script:

<script src="script.js"></script>
